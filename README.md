
#  Smart Crops Protection Information Management System

A web-based Smart Crops Protection Information Management System designed to support farmers with agricultural knowledge, crop protection guidance, weather insights, and smart decision-making tools.

The system integrates AI chatbot, machine learning features, SMS alerts, and email notifications to improve access to agricultural information and enhance farming productivity.

🔗 Live Demo: https://smart-crops-protection-system.vercel.app/



#  Project Overview

This system is designed to help local farmers:
- Access crop protection information before and after harvest
- Receive weather-related farming guidance
- Get real-time agricultural alerts via SMS and email
- Use AI-powered assistant for farming support
- Improve decision-making using data-driven insights


#  Key Features

-  Crop protection information system
-  AI Agent (Chatbot for farming support)
-  SMS notification system
-  Email alert system
-  Machine learning-based recommendations
-  Weather-related farming guidance
-  Farmer-friendly dashboard
